EPITOME (Enhanced Phylogenetic Inference Through Optimized Mapping Efficiency)

Key Features

EPITOME is a Nextflow pipeline that condenses a diverse set of viral sequences into composite references capturing a dataset's overarching diversity:

  • 🧬Clusters sequences by pairwise genetic distance, then builds composite references (most common nucleotide per position) and/or selects each cluster's centroid
  • 🧬Creates reference sets for consensus-based assembly pipelines like VAPER when distance is based on read-mapping efficiency
  • 🧬Automatically downloads sequence data from NCBI by taxon — or supply your own data
  • 🧬Supports both segmented and non-segmented viral taxa
