VirGO (Viral Genome [Submission File] Orchestration)

Key Features

VirGO is a Nextflow pipeline that orchestrates viral genome submission to NCBI for organisms not supported by fully automated submission:

  • 🧬SeqSender — validates FASTA + metadata and generates submission files (.src, .sbt), with optional automated BioSample submission over FTP
  • 🧬VADR — annotates genomes and trims terminal N's
  • 🧬table2asn — produces submission-ready .sqn files for GenBank
  • 🧬Works with any viral pathogen that has a VADR model — tested on Measles and West Nile virus (not for flu A/B or SARS-CoV-2, which submit automatically)
